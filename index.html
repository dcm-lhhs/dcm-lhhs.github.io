About MasterMindAI

MasterMindAI is a chess engine developed completely from scratch using the Python programming language. The main goal of my project is providing the community with code they can use as a reference for chess programming, discussing improvements on my engine and your engines, and learn together!

Downloads

MasterMindAI uses the UCI chess protocol to receive and send moves. Since it would be really boring to type the commands manually, the easiest way to use this engine is by means of a GUI (graphical user interface).


Unfortunately, MasterMindAI can't analyse games at the moment (feel free to contribute to the source code to make this happen!). Its only functionality right now is playing games, and you can test it on the website lichess.org.

Lichess instructions

It's really easy to play MasterMindAI in lichess.org! Just create an account here if you don't have one already. Log in, go to the MasterMindAI lichess profile, press the challenge button, set a time and a color and you're good to go!
